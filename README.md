## Features

- [x] Add child  
- [x] Add parents  
- [x] Edit and delete members      
- [x] Visual family tree rendering  
- [x] Persistent storage with SQLite
- [x] Email family tree link
- [x] Authentication
- [x] Cookies auth session
- [x] Cookies track last visit time
- [ ] Export family tree  
- [ ] Import existing family tree  
- [ ] Multi-user real-time collaboration
